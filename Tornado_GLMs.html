def graphs(x,y,data,model):
    """able to graph regression plot on scatter plot with actual values and fitted values obtained from model"""
    data['fit_values']= model.fittedvalues
    sns.regplot(x=x, y=y, data=data, line_kws={'color':'black', 'label':'LM fit'})
    plt.title('Actual vs Fitted Plot')
    sns.scatterplot(x=x, y='fit_values', data=data, color='red', label='Poisson')

Interpretations: interpreting the coefficients (beta) isn't as straightforward as it is with linear regression because of the multiplicative effect induced by the link function. We can compute the substantive effects by using numpy's built-in exp method.

  • Magnitude: For each unit increase in the Magnitude of a tornado, the predicted number of fatalities is expected to multiply by approximately 6.0535 times, holding other variables constant.
  • Length: For each mile increase in the Length of a tornado track, the predicted number of fatalities is expected to multiply by approximately 1.0135 times.
  • Width: For each yard increase in the Width of a tornado, the predicted number of fatalities is expected to multiply by approximately 1.0001 times.

All of these variables are statisitically significant for the model. Unsurprisingly there was a positive relationship between the magnitude of a tornado and the number of fatalities.

However, it's important to note that the model's predictive power, as indicated by the pseudo R-squared value of 0.4215, suggests a moderate fit of the model to the data. While the model provides insights into the relationships between tornado characteristics and fatalities, there may be other factors not included in the analysis that also influence fatality rates in tornadoes.
